## 💻 Códigos e Citações

Se você precisar destacar um bloco longo de código, a sintaxe de crases fornece _syntax highlighting_ automático:

```typescript
// Exemplo de bloco de código com Syntax Highlighting
interface User {
  id: string;
  name: string;
  role: "admin" | "user";
}

const getUser = async (id: string): Promise<User> => {
  const res = await fetch(`/api/users/${id}`);
  return await res.json();
}
```
